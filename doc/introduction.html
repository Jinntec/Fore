Introduction

"fore" - something that occupies a front position"
- Merriam Webster dictionary

Fore is a declarative, model-driven forms framework.

Main features are:

  • MVC architecture
  • declarative HTML syntax
  • binding facility to seemlessly work with XML data in HTML
  • UI fully implemented in Web Components

Fore is related to XForms 2.0 taking the major ideas of it but not trying to be a complete implementation. Instead the development will be driven by user request, adding features as they become relevant.
