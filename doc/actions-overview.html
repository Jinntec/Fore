Actions

Actions are a declarative way to change state and are triggered by events.

By using a declarative set of actions the need for scripting is massively reduced while providing better readability and maintainence.

Action Overview

Action Description
fx-action allows to group a set of actions
fx-append appends an entry to a fx-repeat
fx-delete deletes one entry from a repeat
fx-dispatch dispatches an event
fx-insert insert an item into a fx-repeat
fx-load load a document into the browser
fx-message display a message to the user
fx-refresh forces a full UI update reapplying the UI Model to the controls
fx-reset roll back all data instances to their initial state after init
fx-return returns values from an embedded form (fx-control)
fx-setindex set the index of a fx-repeat
fx-setvalue set the value of an data item
fx-submit triggers execution of a fx-submission
fx-toggle toggles a specific fx-base
fx-update force complete update of UI by passing all changes to server
