Third-party autocomplete with datalist-ajax

This demo shows the use of a third-party component - datalist-ajax - that is able to build a data-list dynamically and use it to pick some value from a list with autocompletion.

The component is loaded from a CDN in this example.

You need to type at least one character into the input to see the filtered list.
