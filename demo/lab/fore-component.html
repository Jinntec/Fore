Wrapping Fore pages into a web component

constructable CSSStylesSheets are new and need to be polyfilled on some browsers like Safari.

If you prefer to encapsulate Fore pages into shadowDOM or modularize your page and hiding complex sub-sections you can use the generic fore-component. This takes a link to the page to load plus optionally some link elements to inject CSS into the shadowDOM.
